# Localization Readiness

Launch locale: `en-US`.

Planned future locales include Spanish, French, Brazilian Portuguese, Hindi, Arabic, Simplified Chinese, Japanese, Korean, German, Swahili, and Yoruba.

Rules:

- Avoid slang in core UI labels.
- Keep buttons short.
- Use plain sentences.
- Do not hardcode culturally specific examples in global UI.
- Keep music genre labels flexible by region.
- Support right-to-left layouts in future.

Do not add a heavy translation framework until product copy stabilizes.
