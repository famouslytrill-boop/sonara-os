# Lyric Structure Engine™

The Lyric Structure Engine™ takes lyrics the user already wrote and maps them into a workable music structure.

It returns:

- suggested song form
- hook candidates
- verse/pre-hook/bridge candidates
- section plan
- breath markers
- repetition map
- rhyme and cadence notes
- emotional arc
- missing section warnings
- revision checklist
- performance notes

Rules:

- No OpenAI required.
- Short lyrics get a simple structure instead of invented verses.
- User language is preserved when explicitness mode allows it.
- Clean/radio-safe modes warn on profanity and suggest cleaner alternatives.
- No hateful, illegal, exploitative, or copied lyrics.
