# In-App Browser Visual Review

Route:

- `/dashboard/visual-review`

## Local Review

1. Run `npm run dev`.
2. Open the local URL shown by Next.js.
3. Check:
   - `/`
   - `/dashboard/entities`
   - `/dashboard/entities/parent-company`
   - `/dashboard/entities/creator-music-technology/browser`
4. Resize to phone, tablet, and desktop widths.
5. Confirm setup-required labels are visible and not pretending to be live features.

Report UI issues as proactive actions on the relevant entity.
