# Research & Development Roadmap

SONARA should launch with stable, lightweight systems and document heavier work for later.

## Now

- local rules engine
- runtime target engine
- prompt length engine
- slider recommendations
- sound rights system
- branded exports
- Stripe web monetization
- PWA install support
- OBS-ready broadcast kit

## Later

- local LLM provider
- Ollama / LM Studio integration
- Essentia.js deeper audio analysis
- librosa microservice
- FFmpeg deeper metadata extraction
- OBS WebSocket integration
- Capacitor Android wrapper
- Google Play Billing
- Apple IAP
- advanced marketplace
- team collaboration
- real-time co-writing rooms

Later items should stay documented or safely placeholdered until they are real launch priorities.
