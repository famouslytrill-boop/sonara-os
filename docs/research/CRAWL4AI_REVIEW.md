# Crawl4AI Review

Crawl4AI-style crawling is a future adapter concept for public and user-authorized research intake. It is not installed or wired as a live crawler in this sprint.

Allowed future use:

- User-owned websites
- Public docs
- Public GitHub READMEs
- Public product pages
- Government or public data sites where terms permit
- User-provided URLs with explicit permission

Blocked:

- Paywall bypass
- Private data
- Login-required data without explicit owner permission
- Mass scraping social platforms
- Scraping personal profiles
- Credential harvesting
- Spam list creation
- Robots.txt violation
- Terms-of-service bypass

Before implementation, SONARA needs queueing, robots handling, rate limits, storage/RLS review, abuse monitoring, and legal review.
