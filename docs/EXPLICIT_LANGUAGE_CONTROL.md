# Explicit Language Control

Default mode: `radio_safe`.

Modes:

- `clean`: no profanity or graphic content.
- `radio_safe`: mild slang allowed; profanity should be softened.
- `mature`: profanity and adult themes allowed when natural, with safety limits.
- `explicit`: stronger language allowed, but no hateful slurs, illegal instructions, targeted harassment, exploitative sexual content, or graphic sexual content involving minors.

Exports must include `explicitnessMode`.

The control is a label and guidance system; it does not guarantee distribution, radio, playlist, or brand approval.
