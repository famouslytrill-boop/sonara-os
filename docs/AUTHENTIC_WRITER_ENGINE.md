# Authentic Writer Engine™

The Authentic Writer Engine™ helps creators write with more specificity, context, and vocal truth.

## Uses

- English craft
- journalism
- field reporting
- social observation
- psychology
- vocal performance
- music storytelling
- brand copy

## Required Detail Logic

- concrete object
- specific place
- emotional contradiction
- sensory detail
- imperfect admission
- spoken line
- target audience
- social context

## Boundaries

This is not a fake biography generator, trauma farm, direct artist imitation tool, or celebrity cloning system. If input is minimal, it returns reporting questions instead of inventing fake details.
