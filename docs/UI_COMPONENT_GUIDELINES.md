# UI Component Guidelines

- Use bento grids for grouped operating-system surfaces.
- Use soft dark cards with visible borders.
- Use gradient or purple CTAs for primary actions.
- Use dark secondary buttons.
- Keep labels visible.
- Keep buttons at least 44px tall.
- Keep mobile first.
- Keep trust, support, privacy, terms, and contact links available.
- Do not hide important setup blockers behind decorative UI.
