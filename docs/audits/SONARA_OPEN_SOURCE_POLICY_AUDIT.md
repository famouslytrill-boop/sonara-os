# SONARA Open Source Policy Audit

## Scope

This audit covers the public open-source promise, Research Lab watchlist, legal open-source policy, and third-party tool review posture.

## Public Surfaces

- `/open-source`
- `/open-source/philosophy`
- `/open-source/third-party-notices`
- `/open-source/license-policy`
- `/research-lab/open-source`
- `/legal/open-source-policy`

## Current Position

SONARA is free to start and open-source-aware by design. Premium plans pay for hosted infrastructure, higher limits, automation, collaboration, business-grade workflows, support, admin controls, and setup services.

## Free Plan Positioning

- Basic public profile/page
- Basic setup wizard
- Basic templates
- Basic Research Lab visibility
- Basic Creator Tool Library visibility
- Limited records
- No required payment setup for basic public browsing
- No required third-party API key for basic public browsing

## Paid Tier Positioning

- Hosted infrastructure
- Higher limits
- Billing/payment workflows
- Collaboration
- Automation
- Advanced Research Lab features
- Business records
- Admin controls
- Support
- Setup services

## What Free Does Not Mean

- Third-party provider costs may apply if enabled.
- Stripe, Supabase, OpenRouter, email, SMS, storage, and model-provider usage may require separate credentials or costs.
- Premium setup services remain optional paid services.
- Open-source references are not automatically bundled.
- Restricted or non-commercial model weights are not included in paid product features without legal approval.

## License Gates

- Allowed: MIT, BSD, Apache-2.0, and other permissive licenses after attribution check.
- Review Required: MPL, LGPL, unclear dual licenses, model licenses, and complex dependency chains.
- Restricted: GPL and AGPL for proprietary app code without legal approval.
- Blocked: non-commercial model weights in paid features, unknown-license assets, private assets, and copyrighted assets without rights.

## Findings

- Public open-source pages exist.
- Research Lab open-source watchlist exists.
- Legal open-source policy exists.
- Third-party tools are represented as research/reference candidates unless reviewed.
- No external source code was copied as part of this policy work.

## Human Review Required

- Attorney review before treating this policy as final legal guidance.
- License review before copying, vendoring, installing, self-hosting, or exposing external projects.
- Security review before enabling crawling, automation, model execution, or messaging integrations.
