# Automated Sound Discovery Workflow

Allowed automation:

1. Query metadata.
2. Store title, creator, source URL, license label, and license URL.
3. Run license classifier.
4. Mark export status.
5. Create human review recommendations.

Human approval required:

- downloading files
- storing files in Supabase Storage
- publishing packs
- selling packs
- changing rights category manually
- accepting commercial license proof

Blocked:

- public third-party sound commerce at launch
- resale of third-party downloaded sounds
- automatic rights approval
- unknown license exports
