# Free/Open-Source Automation And GPU Policy

Launch default:

- CPU/local rules.
- No required GPU.
- No required paid model key.
- Supervised autonomy only.

Allowed future GPU research:

- Google Colab prototypes.
- Kaggle notebooks.
- Local NVIDIA CUDA workstation if available.
- Cloud GPU only when budget exists.

Automation boundaries:

- No uncontrolled self-modifying code.
- No automatic publication.
- No automatic sound downloading for resale.
- No automatic env var changes.
- No automatic payment configuration changes.
- Risky actions create recommendations and require human approval.
