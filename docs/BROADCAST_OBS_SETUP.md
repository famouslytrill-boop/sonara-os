# Broadcast / OBS Setup

SONARA exports an OBS-ready broadcast kit. It does not directly control OBS yet.

## Current Export

The Broadcast Kit generates:

- stream title
- scene list
- OBS scene recommendations
- audio routing notes
- release listening session outline
- live premiere checklist
- visual overlay suggestions
- exportable broadcast plan

## Recommended OBS Workflow

1. Create scenes for starting soon, host welcome, project context, first listen, breakdown, call-to-action, and closing slate.
2. Add separate audio sources for microphone and music playback.
3. Keep lower-third text readable on mobile.
4. Run a private recording test before the premiere.
5. Save the recording for clips and archive use.
