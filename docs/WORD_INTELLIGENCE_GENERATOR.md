# Word Intelligence Generator

The Word Intelligence Generator helps users find:

- stronger words
- hook phrases
- rhyme-adjacent words
- clean alternatives
- explicit alternatives when enabled
- metaphor seeds
- regional phrasing notes
- genre-aware vocabulary
- source and attribution notes
- revision prompts

Rules:

- Must work offline with local rules.
- Must not require OpenAI.
- Must not require live API calls to render the app.
- Must respect explicitness mode.
- Must not generate hateful slurs.
- Must not copy definitions as if original.
- Must not bulk scrape Urban Dictionary.
- Must treat Urban Dictionary as manual-review slang signal only.
- Wikimedia/Wikipedia/Wiktionary lookup is optional and source-aware.
- Store source title, URL, license note, and retrieval date when external sources are used.
- Do not bulk-download external data.
