# Supervised Autonomy System

SONARA uses supervised autonomy, not unsupervised production changes.

## Levels

- off
- assistive
- supervised
- autonomous_safe

Default: `supervised`.

## Rules

- Safe checks may run automatically.
- Risky actions create recommendations only.
- Payment changes require human approval.
- Legal/trademark changes require human approval.
- Deleting data requires human approval.
- Publishing store products requires human approval.
- Updating environment variables requires human approval.

## Background Checks

Checks cover brand consistency, trademark symbols, private artist name scans, export footers, sound rights, runtime, prompt length, sliders, Authentic Writer, vector memory, mobile/PWA, monetization, Supabase, Vercel env safety, store readiness, OBS kit, docs freshness, activation, conversion, trust/support, launch campaign, founder command center, open-source stack, IPO-grade discipline, and moat strength.
