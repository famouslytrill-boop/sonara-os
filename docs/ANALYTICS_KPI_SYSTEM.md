# Analytics KPI System

SONARA should track the business with IPO-grade discipline, without claiming it is IPO-ready.

## KPIs

- visitors
- signups
- activated users
- projects created
- exports created
- store visits
- checkout starts
- paid conversions
- churn risk
- failed builds/manual setup blockers

## Readiness Scores

Founder Command Center summarizes launch, product, company, marketing, profit, monetization, store, PWA, safe autonomy, and moat readiness.
