# Native App Future

Native apps are a later packaging layer, not a product expansion excuse.

Native work should be limited to:

- Better offline capture.
- Studio session workflow improvements.
- Push notifications with explicit user consent.
- File handling for creator-owned assets.
- App store distribution after the PWA is stable.

Native work should not introduce fully autonomous public posting, unlimited heavy generation, unauthorized voice/likeness cloning, or claims about guaranteed outcomes.
