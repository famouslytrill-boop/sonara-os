# Release Readiness Score

The Release Readiness Score is a local-rule score from 0 to 100.

Areas:

- lyrics structure
- arrangement clarity
- style prompt clarity
- metadata completeness
- rights safety
- export completeness
- brand consistency
- release strategy
- authenticity signals

Statuses:

- blocked
- needs_work
- ready
- strong

The score helps creators prioritize next steps. It does not guarantee streams, royalties, playlist placement, distribution approval, or commercial success.
