# Pre-Launch Required Work

Before public customer launch, SONARA must complete these required gates:

1. Owner, staff, and customer permission enforcement.
2. Real CRUD APIs per active module.
3. Stripe live payment event verification.
4. Resend sending-domain verification.
5. Supabase Storage bucket setup.
6. Realtime access rules.
7. Admin control-plane visualizer expansion.
8. Mobile, tactile, and premium UI layer.

A gate is complete only when it is implemented, tested, visible in admin readiness, and has a clear Setup Required fallback when configuration is missing.
