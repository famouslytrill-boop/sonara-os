# Store Product Readiness

Store products must pass a readiness gate before paid publishing.

## Checks

- product title exists
- product description exists
- price exists
- Stripe price ID exists for paid product
- license terms exist
- preview image exists or placeholder exists
- included files list exists
- refund/support note exists
- rights classification exists
- export bundle exists

## Statuses

- draft
- ready
- blocked
- needs_payment_setup
- needs_license_review
