# Arrangement Core System

The Arrangement Core turns a user idea into practical structure:

- intro strategy
- verse strategy
- hook strategy
- bridge strategy
- outro strategy
- transition notes
- energy curve
- vocal placement
- drum movement
- arrangement risks
- genre authenticity notes

Inputs can include genre family, subgenre, mood, BPM, key, runtime target, vocal mode, drum language, harmonic identity, release goal, platform goal, and arrangement complexity.

The engine is local-rules only. It integrates with Genre Universe, Runtime Target, Prompt Length, Authentic Writer, Lyric Structure, Sound Identity, External Generator Settings, and exports.
