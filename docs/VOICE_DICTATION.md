# Voice Dictation

`VoicePromptInput` provides browser-compatible dictation readiness using the Web Speech API when available.

## Privacy

- Microphone permission is controlled by the browser/device.
- Transcript text stays local until the user submits it.
- Do not dictate secrets, credentials, payment data, or private sensitive information into notes.

## Support

Browser support varies. If unavailable, users can type the same note manually.
