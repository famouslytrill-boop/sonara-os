# Sound Discovery System

SONARA uses metadata-first, rights-safe sound discovery.

Supported source types:

- Freesound API
- Openverse API
- Wikimedia Commons API
- Internet Archive metadata/search API
- user uploaded assets
- manually curated open collections

Launch rules:

- Discover and classify metadata first.
- Do not bulk-download sound files automatically.
- Do not publish or sell sound packs without human rights approval.
- Open-license data must be verified per item.
- User uploads require ownership/license attestation.

Product rule:

SONARA can discover the world's open sounds. SONARA cannot sell the world's open sounds until rights are verified.
