# PWA And App Store Preparation

PWA support includes:
- `app/manifest.ts`
- service worker at `/sw.js`
- offline page
- install prompt

Do not cache:
- Stripe Checkout
- webhooks
- sensitive API responses
- private dashboards

App store preparation:
- final icons
- screenshots
- privacy policy URL
- support URL
- description
- age/content rating notes
- data collection notes
- native billing review for mobile app stores if subscriptions are sold inside native apps
